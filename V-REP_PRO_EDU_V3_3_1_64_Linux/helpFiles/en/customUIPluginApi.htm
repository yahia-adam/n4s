Qt-based custom UI API reference

The list of API functions below provides an interface to the Qt-based custom UI plugin for V-REP.

simExtCustomUI_create

Description Create a window.
See also: simExtCustomUI_destroy
Lua synopsis int uiHandle=simExtCustomUI_create(string xml)
Lua parameters
xml: xml ui definition
Lua return values
uiHandle: a handle to the created UI

simExtCustomUI_destroy

Description Destroy a window.
See also: simExtCustomUI_create
Lua synopsis simExtCustomUI_destroy(int handle)
Lua parameters
handle: ui handle
Lua return values -

Lua synopsis int value=simExtCustomUI_getCheckboxValue(int handle, int id)
Lua parameters

simExtCustomUI_insertComboboxItem

Description Insert an item in a combobox widget.
Lua synopsis simExtCustomUI_insertComboboxItem(int handle, int id, int index, string text)
Lua parameters
handle: ui handle
id: widget id
index: If the index is equal to or higher than the total number of items, the new item is appended to the list of existing items. If the index is zero or negative, the new item is prepended to the list of existing items.
text: Item text.
Lua return values -

simExtCustomUI_isVisible

Description Check if the window is visible.
Lua synopsis bool visibility=simExtCustomUI_isVisible(int handle)
Lua parameters
handle: ui handle
Lua return values
visibility: true if the window is visible, false otherwise
