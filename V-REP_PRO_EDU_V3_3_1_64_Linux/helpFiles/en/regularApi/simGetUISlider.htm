Regular API function

simGetUISlider
remote API equivalent: simxGetUISlider
RosPlugin API equivalent: simRosGetUISlider

Description Gets the slider position of an OpenGl-based custom UI button (must be slider-type button). See also simSetUISlider.
C synopsis simInt simGetUISlider(simInt uiHandle,simInt buttonHandle)
C parameters
uiHandle: handle of an OpenGl-based custom UI
buttonHandle: handle of a button inside the specified OpenGl-based custom UI
C return value
slider position (value between 0 and 1000) or -1 if operation was not successful
Lua synopsis number sliderPosition=simGetUISlider(number uiHandle,number buttonHandle)
Lua parameters
Same as C-function
Lua return values
Same as C-function
