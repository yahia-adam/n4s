simGetShapeMaterial

Description Retrieves the material ID of a specific shape. See also simGetMaterialId and simSetShapeMaterial.
C synopsis simInt simGetShapeMaterial(simInt shapeHandle)
C parameters
shapeHandle: the handle of the shape. See also simGetObjectHandle.
C return value
The material ID, or -1 n case of an error

Same as C-function
