simGetUIProperty

Description Retrieves the properties of an OpenGl-based custom UI. See also simSetUIProperty.
C synopsis simInt simGetUIProperty(simInt uiHandle)
C parameters
uiHandle: handle of an OpenGl-based custom UI
C return value
OpenGl-based custom UI property values, -1 if operation was not successful

Lua parameters
Same as C-function
