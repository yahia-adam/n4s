Regular API function

simUnpackInts

Description Unpacks a string (or part of it) into a table of integer numbers. See also simPackInts, simUnpackUInts, simUnpackFloats, simUnpackDoubles, simUnpackWords and simUnpackBytes.

C parameters

Lua synopsis table integerNumbers=simUnpackInts(string data,number startIntIndex=0,number intCount=0,number additionalCharOffset=0)
Lua parameters
data: a string (values between 0 and 255) that contains packed integer numbers
startIntIndex: the zero-based index from which on data should be unpacked (from data[4*startIntIndex+1+additionalCharOffset]). Can be omitted in which case 0 is used
intCount: the amount of ints that should be unpacked. Can be omitted in which case 0 is used (which indicates that the maximum of ints should be unpacked from the indicated startIntIndex)
additionalCharOffset: a char offset that will be added before reading the ints. Can be omitted, in which case 0 is used.
Lua return values
integerNumbers: a table containing unpacked integer numbers if operation was successful, nil otherwise
